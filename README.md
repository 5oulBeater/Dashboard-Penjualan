# Dashboard Penjualan Laravel

## Cara Install Lokal
git clone https://github.com/5oulBeater/Dashboard-Penjualan.git
cd Dashboard-Penjualan
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
